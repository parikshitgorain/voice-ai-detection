# Contributing

Thanks for your interest in contributing!

## Quick Guidelines
- Keep changes focused and production-ready.
- Avoid adding hardcoded secrets or environment-specific paths.
- Update docs if behavior changes.

## Development Setup
1) Install backend dependencies:
```
cd backend
npm install
```
2) Python venv for deep model inference:
```
python3 -m venv deep/.venv
./deep/.venv/bin/pip install -r deep/requirements.txt
```
3) Run API:
```
node server.js
```
4) Run frontend:
```
python3 -m http.server 5173 --directory frontend
```
