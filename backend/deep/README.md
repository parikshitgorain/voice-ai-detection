# Deep Model

Python-based deep model training and inference.

Key scripts:
- `train_multitask.py` multitask trainer
- `infer_multitask.py` inference runner
- `generate_ai_tts.py` optional TTS data generation helper

Trained models (tracked in repo):
- `multitask_English.pt`
- `multitask_Hindi.pt`
- `multitask_Tamil.pt`
- `multitask_Malayalam.pt`
- `multitask_Telugu.pt`

If running inference here, create a venv and install:
```
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
```
